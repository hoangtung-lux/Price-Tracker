# Price Tracker Pro v3.1.7 (Guardian)

Công cụ theo dõi lịch sử giá mã nguồn mở, bảo mật và thẩm mỹ cho các sàn TMĐT Trung Quốc (Taobao, Tmall, JD, 1688).

![Price Tracker Preview](https://raw.githubusercontent.com/hoangtung-lux/Price-Tracker/main/assets/logo.png)

## ✨ Tính năng nổi bật
- **Theo dõi lịch sử giá**: Biểu đồ trực quan, chính xác.
- **Săn mã giảm giá**: Tự động phát hiện và chèn mã giảm giá "ẩn" từ Manmanbuy.
- **Giao diện Pro Max**: Thiết kế kính mờ (Glassmorphism), hỗ trợ tối ưu trải nghiệm người dùng.
- **Bảo mật Guardian**: Hệ thống mã hóa đa tầng bảo vệ dữ liệu API.

## 📘 Hướng dẫn cài đặt nhanh (Cho người mới)

Dành cho bạn nào chưa biết cách cài đặt, hãy làm theo 2 bước đơn giản sau:

### Bước 1: Cài đặt tiện ích hỗ trợ
Bạn cần cài bộ máy chạy script tên là **Violentmonkey** vào trình duyệt:
- [Cài Violentmonkey cho Chrome / Edge / Cốc Cốc](https://chrome.google.com/webstore/detail/violentmonkey/jinjaccaljkdabocponuonmfmeneiojj)
- [Cài Violentmonkey cho Firefox](https://addons.mozilla.org/vi/firefox/addon/violentmonkey/)

### Bước 2: Cài đặt Price Tracker
Sau khi đã xong Bước 1, bạn nhấn vào link dưới đây để cài đặt script:

👉 [**NHẤN VÀO ĐÂY ĐỂ CÀI ĐẶT PRICE TRACKER**](https://raw.githubusercontent.com/hoangtung-lux/Price-Tracker/main/price-tracker.user.js)

*(Một trang web sẽ hiện ra, bạn nhấn nút **"Confirm Installation"** màu xanh để hoàn tất).*

# PROJECT: Price Tracker Pro Max (Việt Hóa)

## Vision
The definitive e-commerce price tracking solution for the Vietnamese community, supporting Taobao, Tmall, JD, and 1688 with professional visualizations and secure credential management.

## Goals
- Multi-platform support (Taobao, Tmall, JD, 1688).
- ECharts-powered price history visualizations.
- Professional obfuscation and vault-based credential protection.
- High-availability update mirrors via GitHub/Gitee.

## Tech Stack
- **Core**: Tampermonkey Userscript (JavaScript).
- **Security**: Python-based obfuscation, XOR Vault encryption.
- **Visuals**: ECharts 5.5.0.
- **Infrastructure**: Python minification and build scripts.
